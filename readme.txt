The following MATLAB functions are a small collection of complexity measures for one-dimensional signals which are used in the following papers:

Burns T & Rajan R (2019) A mathematical approach to correlating objective spectro-temporal features of non-linguistic sounds with their subjective perceptions in humans. Frontiers in Neuroscience, 13:794.
Burns T & Rajan R (2015) Combining complexity measures of EEG data: multiplying measures reveal previously hidden information. F1000Research, 4:137.
If you use these codes, please reference the above work(s).
